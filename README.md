## Commodity Trading Rules AI

An interactive web app for exploring commodity trading rules, regulations, and procedures via a chat interface with document citations.

### Getting started

**Install dependencies**

```sh
npm install
```

**Run the dev server**

```sh
npm run dev
```

The app uses **Vite + React + TypeScript + shadcn-ui + Tailwind CSS**.

### Project scripts

- **`npm run dev`**: start the development server
- **`npm run build`**: create a production build
- **`npm run preview`**: preview the production build locally
- **`npm run lint`**: run ESLint
- **`npm run test`**: run tests once
- **`npm run test:watch`**: run tests in watch mode

### Tech stack

- **Vite** for bundling and dev server
- **React + TypeScript** for the frontend
- **shadcn-ui + Tailwind CSS** for UI components and styling
